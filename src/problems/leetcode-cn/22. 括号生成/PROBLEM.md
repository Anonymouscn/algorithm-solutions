# 22. 括号生成

<p style="color: #ffb800; display: inline-flex; item-align: center; justify-content: center; padding: 2px 10px; border: 2px solid #666; border-radius: 20px">中等</p>

数字 `n` 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 **有效的** 括号组合。


**示例 1：**

> **输入：** n = 3 \
> **输出：** ["((()))","(()())","(())()","()(())","()()()"]


**示例 2：**

> **输入：** n = 1 \
> **输出：** ["()"]


**提示：**

* `1 <= n <= 8`
